
## About OCRusion
OCRusionはオンラインでOCR処理をするWEBサービスです。
GoogleのCloud Vision APIを利用して、OCR処理を行っています。
期待通り動かなかった、もっと使いやすくしてほしいことがあればIssueなどでご連絡ください。
